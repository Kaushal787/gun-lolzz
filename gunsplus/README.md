gunsplus
========

Monorepo for a guns.lol-style platform with a Next.js (App Router) web app.

Apps
- apps/web: Next.js 14 (App Router) + Tailwind CSS

Scripts
- dev: Run app in dev
- build: Build app
- start: Start production server

Getting Started
1. Install deps: `npm install`
2. Dev: `npm run dev --workspace @gunsplus/web`
3. Open http://localhost:3000
